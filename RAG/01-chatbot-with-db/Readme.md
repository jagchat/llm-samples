# ChatBot using RAG with a database

NOT WORKING - work in progress

This is a sample of how to use the RAG library in a project that uses a database.

## Prerequisites

- Docker
- Python

## How to run

- run `\start.ps1`
- run `\embeddings_updater\main.py`
- run `\chatbot\main.py`

## How to stop

- chatbot - Ctrl+C
- embeddings_updater - Ctrl+C
- run `\stop.ps1`
